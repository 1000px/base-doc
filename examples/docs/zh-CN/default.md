### KFront-BaseUI

#### KFront-BaseUI发布日志

2018-07-10

	kfront-baseui@1.0.1

#### 1. 综述

KFront-BaseUI是一款基于Vue.js(2.*)与ElementUI开发定制的UI解决方案。

开发者可在KFront-Base体系内使用该UI库。

根据公司业务项目需求，提炼出更凝练的交互设计和视觉风格。

开箱即用的Vue.js(2.*)组件库。

#### 2. 设计原则

一致性

提高开发效率

简化操作流程

样式可扩展

#### 3. 快速上手

进入项目根目录，运行下述命令，安装依赖

```
npm install --save kfront-baseui@latest
```
